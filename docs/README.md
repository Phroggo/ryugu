# Project Documentation

Complete documentation set for the SpaceHopper Ryugu simulation, kept in sync
with the code it describes.

- `Research_Paper.md` — the academic paper (design, mathematics, and measured
  verification results)
- `research_report.md` — the technical study companion: how the environment
  was built, the robot's full design derivations, and engineering case studies
  (the sensor-bridge failure, micro-gravity landing detection, the liftoff
  campaign, and more)
- `HANDOFF.md` — the engineering logbook and continuation guide: environment
  gotchas, commit-by-commit history, and the project checklist (read this
  first when resuming work)
- `task.md` — checkbox-level task tracker (Phases A–J)
- `walkthrough.md` — narrative development log
- `heightmap_preview.png`, `spacehopper_concept_*.png` — figures referenced by
  the papers
